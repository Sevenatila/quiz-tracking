# Downloads Directory

This directory contains the digital products that customers can download after purchase.

## Files Structure:

- **plantilla-finanzas-es.xlsx** - Main Spanish financial template
- **plantilla-finanzas-pt.xlsx** - Main Portuguese financial template
- **bonus-guias.zip** - Bonus guides pack
- **video-tutorial.zip** - Video tutorial package
- **soporte-premium.pdf** - Premium support guide

## Security Note:

In production, consider:
- Moving files to a private storage (AWS S3, etc.)
- Using signed URLs for downloads
- Implementing access control
- Setting download expiration times
